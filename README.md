## 0.KimiBoardについて 
>[!WARNING]
>**・ロープロファイルのキーボードには対応しておりません。**  
>**・スペースキーのキースイッチを取り外して使用しますが、取り付けるキーボードによっては支え用パーツやLEDなども取り外す必要がございます。**  
>**・スペースバー埋め込みパーツは全てのキーボードに対応しているわけではありません。**  
>**・商品ページのサイズ詳細などを元に、お手元のキーボードのサイズを測り事前にご確認の上ご購入をお願いします。**
